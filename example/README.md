# Timeline Example App

This example demonstrates how to use the `swiip_pubdev_timeline` package in a Flutter application.

## Setup

1. Copy `.env.sample` to `.env` and configure your API credentials:
   ```bash
   cp .env.sample .env
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the example:
   ```bash
   flutter run
   ```

## Features Demonstrated

- Fetching timeline data from an API
- Configuring timeline colors and appearance
- Handling user interactions (day clicks, stage edits, element edits)
- Performance optimization with large datasets

## Configuration

The example uses environment variables for API configuration. See `.env.sample` for required variables.
